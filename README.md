# Ilskehanteraren

Step-by-step anger management for children with NPF/autism.

## Features
- 😤 Anger level check-in (5 levels)
- 🌬️ Multiple calming strategies (breathing, walking, counting, squeezing, talking)
- 📝 Step-by-step guided exercises
- 🌟 Post-exercise reflection
- 🌍 Translatable (gettext)
